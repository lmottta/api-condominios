API de Condomínios

Bem-vindo à API de Condomínios. Esta API fornece endpoints para gerenciar informações sobre condomínios.

Endpoints Disponíveis:

Listar Condomínios

GET /.netlify/functions/api/condominios

Buscar por Bairro

GET /.netlify/functions/api/condominios/busca/bairro/{bairro}

Buscar por Nome

GET /.netlify/functions/api/condominios/busca/nome/{nome}

Buscar por Cidade

GET /.netlify/functions/api/condominios/busca/cidade/{cidade}

Obter Estatísticas

GET /.netlify/functions/api/condominios/estatisticas

Para mais informações, consulte a documentação completa da API.
