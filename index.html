API de Condomínios - Feira de Santana

API REST para gerenciamento de informações de condomínios em Feira de Santana. Acesse a documentação completa através do Swagger ou ReDoc nos links acima.

Endpoints Disponíveis

GET /api/v1/condominios/

Lista todos os condomínios cadastrados

GET /api/v1/condominios/estatisticas

Retorna estatísticas sobre os condomínios

GET /api/v1/condominios/busca/bairro/{bairro}

Busca condomínios por bairro

GET /api/v1/condominios/busca/nome/{nome}

Busca condomínios por nome

POST /api/v1/condominios/

Cria um novo condomínio

PUT /api/v1/condominios/{uuid}

Atualiza um condomínio existente

DELETE /api/v1/condominios/{uuid}

Remove um condomínio

Exemplo de Uso

curl -X GET "https://api-condominio.netlify.app/api/v1/condominios/estatisticas"

Resposta:

{
    "total_condominios": 1000,
    "condominios_por_bairro": {
        "SIM": 85,
        "Santa Mônica": 72,
        "Tomba": 65,
        ...
    }
}
